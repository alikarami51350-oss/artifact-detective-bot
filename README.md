# کارآگاه باستانی 🏺 — ربات تلگرامی تشخیص شی باستانی

این ربات از کاربر چند عکس از یک شی از زوایای مختلف می‌گیره، چند سوال زمینه‌ای
می‌پرسه، و با استفاده از **Gemini (هوش مصنوعی رایگان گوگل)** تحلیل می‌کنه که
آیا احتمالاً ساخته‌ی دست بشره یا پدیده‌ی طبیعیه، و در صورت انسان‌ساخت بودن
حدس می‌زنه برای چه هدفی ساخته شده.

## پیش‌نیازها

- Python 3.10 یا بالاتر
- یک توکن ربات تلگرام (از [@BotFather](https://t.me/BotFather))
- یک کلید API **رایگان** از [Google AI Studio](https://aistudio.google.com/apikey)
  برای Gemini — نیازی به کارت بانکی یا شارژ حساب نیست، فقط کافیه با یک
  حساب گوگل وارد بشید.

## نصب

```bash
cd artifact-detective-bot
python -m venv venv
source venv/bin/activate   # در ویندوز: venv\Scripts\activate
pip install -r requirements.txt
```

## تنظیم متغیرهای محیطی

فایل `.env.example` رو کپی کنید و مقادیرش رو پر کنید:

```bash
cp .env.example .env
```

سپس مقدار `TELEGRAM_BOT_TOKEN` و `GEMINI_API_KEY` رو داخل `.env` بذارید.

## اجرا

```bash
python bot.py
```

ربات با polling اجرا می‌شه؛ یعنی برای تست محلی نیازی به دامنه یا وبهوک نیست.
کافیه اسکریپت رو اجرا کنید و در تلگرام به ربات پیام `/start` بدید.

## روند مکالمه

1. کاربر با `/start` شروع می‌کنه.
2. چند عکس (حداقل ۲-۳ تا) از شی می‌فرسته و در پایان روی دکمه‌ی
   «✅ عکس‌ها تمام شد» می‌زنه.
3. ربات به ترتیب چند سوال زمینه‌ای می‌پرسه (محل پیدا شدن، جنس، اندازه،
   نشونه‌های کنده‌کاری، اشیای اطراف، حدس شخصی کاربر).
4. عکس‌ها + پاسخ‌ها با هم به مدل Gemini فرستاده می‌شن و تحلیل نهایی
   برای کاربر ارسال می‌شه.

## نکات مهم

- تحلیل انجام‌شده صرفاً یک حدس هوش مصنوعیه و **جایگزین نظر باستان‌شناس
  واقعی نیست**. در متن پاسخ هم همیشه این نکته به کاربر یادآوری می‌شه.
- عکس‌ها به‌صورت موقت روی دیسک (`/tmp`) ذخیره و بعد از تحلیل پاک می‌شن.
- سطح رایگان Gemini محدودیت تعداد درخواست در روز داره (برای استفاده‌ی
  شخصی و تست کاملاً کافیه)، و ممکنه گوگل از داده‌های این سطح برای
  بهبود مدل‌هاش استفاده کنه — پس عکس/اطلاعات کاملاً محرمانه نفرستید.
- برای جابجایی به سرور واقعی، بهتره از `run_polling` به webhook (مثلاً با
  FastAPI یا خود `python-telegram-bot`) مهاجرت کنید.

## سفارشی‌سازی

- سوالات داخل لیست `QUESTIONS` در `bot.py` قابل ویرایش هستن.
- برای تغییر مدل Gemini، مقدار `GEMINI_MODEL` رو در `.env` عوض کنید.
- برای تغییر لحن یا ساختار تحلیل، متغیر `system_prompt` داخل تابع
  `analyze_and_reply` رو ویرایش کنید.
